# Treasury

The Treasury Deposit Event

- ## TreasuryDepositEvent
	- **Summary:** Some funds have been deposited.
	- **Datas returned:** 
  		- account: *string as AccountId32*
		- to: *string as AccountId32*
  		- value: *string as u128*
  		- valueRounded: *number*
